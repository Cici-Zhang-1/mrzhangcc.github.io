靶心人公式

  1. 主人公的“目标”是什么？
  2. 他的“阻碍”是什么？
  3. 他如何的“努力”？
  4. “结果”如何？（通常是不好的结果）
  5. 如果结果不理想，代表努力无效，那么有超越努力的“意外”可以改变这一切吗？
  6. 意外发生后，情节会如何“转弯”？
  7. 最后的“结局”是什么？

所以靶心人公式：目标 -> 阻碍 -> 努力 -> 结果 -> 意外 -> 转弯 -> 结局
进一步可以拆解为两种公式。
努力人公式： 目标 -> 阻碍 -> 努力 -> 结果
意外人公式： 目标 -> 意外 -> 转弯 -> 结局

三分钟说5040个故事

靶心人公式并不一定按照固定的顺序发生，可以随意排列组合，所以：

$$7 X 6 X 5 X 4 X 3 X 2 X 1 = 5040$$

我们可以在三分钟内说出5040个故事
