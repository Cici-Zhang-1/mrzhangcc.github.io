Mac中iterm快捷键

2018-01-17 技术·工具 iterm

标签

  • ⌘ + t 新建标签
  • ⌘ + w 关闭标签
  • ⌘ + Num || ⌘ + ← || ⌘ + → 切换标签
  • ⌘ + ↩︎ 切换全屏
  • ⌘ + f 查找

分屏

  • ⌘ + d 垂直分屏
  • ⌘ + ⇧ + d 水平分屏
  • ⌘ + ⌥ + ← || ⌘ + ⌥ + → || ⌘ + ⌥ + ↑ || ⌘ + ⌥ + ↓ || ⌘ + [ || ⌘ + ] 切换屏幕
  • ⌘ + ; 查看屏幕历史命令
  • ⌘ + ⇧ + h 查看剪切板历史

其它

  • ⌃ + u 清除当前行
  • ⌃ + e 到行尾
  • ⌃ + a 到行首
  • ⌃ + f || ⌃ + b 前进或者后退
  • ⌃ + p || ⌃ + n 搜索上一条／吓一跳命令
  • ⌃ + r 搜索历史命令
  • ⌃ + d 删除光标所在到字符
  • ⌃ + h 删除光标之前到字符
  • ⌃ + w 删除光标之前到单词
  • ⌃ + k 删除到文本末尾
  • ⌃ + t 交换光标处文本
  • ⌃ + l || ⌘ + r 清屏
