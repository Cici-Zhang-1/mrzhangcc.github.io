PhpStorm支持vue.js语法高亮

2018-01-27 技术·工具 phpstorm

由于最近在做一个项目，需要使用现在比较流行的渐进式框架Vue，而我之前使用的编辑器是PhpStorm，就如下图所示没有对Vue的语法高亮显示。下面介绍如何配置。

开发环境：MacOS + PhpStorm

首先打开PhpStorm配置界面，路径：PhpStorm--->Preferences或者使用快捷键⌘ + ,，入下图所示：

然后在Preferences界面左边菜单栏找到Plugins项：

然后再在右边找到Browse repositories按钮点击，弹出Browse repositories操作界面：

之后在最上面搜索框中输入vue.js找到对应的插件，点击install安装插件。

最后重启PhpStorm编辑器，便会支持Vue语法高亮：
