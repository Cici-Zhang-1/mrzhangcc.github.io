Macbook常用快捷键

2016-06-26 技术·工具 Macbook

切换

command + tab 在应用程序间切换

command + ` 在应用程序中的窗口切换

通用

command + H 隐藏窗口

command + M 最小化窗口

command + N 新建

command + O 打开

command + S 保存

command + shift + S 另存为

command + P 打印

command + W 关闭当前窗口

command + Q 退出应用程序

contrl + command + F 进入/退出全屏

截屏

command + shift + 3 截取整个屏幕

command + shift + 4 截取选择区域

command + shift + 4 + space 截取选择窗口

文件管理器

space 快速查看

command + A 选择全部

command + I 显示简介

command + [ 向后

command + ] 向前

command + ↑ 上层文件夹

command + N 新建文件夹

command + F 搜索

command + C 复制

command + V 粘贴

command + option + V 剪切粘贴

command + delete 删除

command + shift + delete 清空回收站

浏览器

command + = 放大

command + - 缩小

command + T 新建一个选项卡

command + N 新建一个新窗口

command + R 刷新

command + F 搜索

command + D 收藏当前网页

command + W 关闭当前选项卡

系统

control + shift + ⏏ 关闭显示器

command + option + esc 强制退出应用程序

command + option + ⏏ 睡眠

command + control + ⏏ 关闭所有应用程序并重新启动

command + shift + Y 用当前选中的文字生成便签

command + space 切换输入法
